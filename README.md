# JAVA简单爬虫

Description：
---
本爬虫只能用于爬取静态页面

     简单使用方法：
     1.通过实例一个HTMLCrawler，传入网页URL参数
     2.使用getHtmlSource方法可得到String类型的网页内容
     3.使用write方法，传入输出路径，即可输出一个网页源文件
